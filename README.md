# Lectra Workspace

Lectra is the iPad companion workspace under Canvascope.

## What Lives Here

- `Lectra/`: iPad app code + Xcode project.
- `docs/`: functional and technical specs for Lectra + Canvascope integration.
- `design/`: UI explorations, logo exports, and Figma-ready boards.
- `scripts/`: helper scripts for workflow automation.
- `build-prompt.md`: active implementation prompt for coding agents.
- `[FINAL]BUILD_PROMPT.md`: final production-oriented build brief.

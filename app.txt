App Idea (iPadOS): Premium Apple Pencil PDF Annotator (Lectra companion to Canvascope)

Canonical product blueprint reference:
- `docs/lectra-spec.md`

Core concept:
- An iPad-first application designed for heavy Apple Pencil usage.
- Users send PDFs from the Canvascope Chrome extension to their iPad.
- Opening the app shows a grid of available PDFs fetched via Supabase.
- User annotates the PDF with low latency using native PencilKit + PDFKit integration.
- Tapping "Back" or "Done" flattens the strokes onto the PDF and uploads the final version back to Canvascope.

Main user flow:
1. User receives Push/Realtime db update that a new document arrived.
2. User taps the document in the Home Grid. It downloads the raw PDF from Supabase Storage.
3. User annotates heavily (Highlighter, Pen, Eraser).
4. User taps "Sync". App flattens the PKDrawing vectors to the PDF pages.
5. App uploads the overlaid PDF to Supabase Storage and updates the db status to `annotated`.

Design direction:
- Premium Dark mode (or dynamic based on system text).
- Minimalist, glassmorphic UI elements over the PDF to maximize screen real estate.
- Red accent colors (`#E02520`) used sparingly for critical actions or selected tools.

MVP definition:
- Supabase Auth (Google OAuth).
- Home Grid displaying `synced_items`.
- PDFKit + PencilKit editor with accurate coordinate mapping.
- Flatten & Upload background sync loop.
